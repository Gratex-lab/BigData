# BigData
Projekt na przedmiot podstaw przetwarzania dużych zbiorów danych 

Projekt opiera się na integracji danych pochodzących z systemów zewnętrznych oraz ich transformacji do postaci analitycznej.
Dane zostały pozyskane przy użyciu Google Cloud API (np. z publicznych zbiorów NOAA w BigQuery).
Uruchomienie wymaga własnej konfiguracji dostępu do Google Cloud Platform.
